# cookcrew

템플릿으로 폴더 경로 이동 후
1. npm install 또는 yarn install
2. src -> routers -> index.tsx에서 <BrowserRouter> 안에 있는 basename을 지워주거나 변경하세요.
3. package.json에서 homepage field를 나의 URL로 변경해준다

- 설치해야할 것들
4. npm install --save redux
5. npm install --save react-redux
6. npm install --save react-router-dom
7. npm install --save axios
8. npm install --save react-cookie

types.ts(data폴더에)와 index.tsx(router폴더에)에서 url 추가해주면됨
